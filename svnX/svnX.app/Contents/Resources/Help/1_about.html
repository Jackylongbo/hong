1. About svnX

SvnX is a free graphical user interface for most client side features of the Subversion VCS. It helps you to manage & interact with your Subversion working copies, allowing you to select files & directories and perform operations on them. It also helps you with your repositories - browse logs, files, revisions & changes; import, export & checkout; manage the repository’s hierarchy.

Visit the svnX discussion group at <groups.google.com/group/svnx>.

1.1. Features

  • Repository Inspector:
    • Browse logs, revisions and directories in a single window.
    • Svn log, list, diff, checkout, export, import, switch, copy, mkdir, delete and blame support.
    • Drag & drop from the repository browser to the Finder. (svn export or svn checkout)
    • Drag & drop from the Finder to the repository browser. (svn import)
    • Drag & drop between a repository and a working copy. (svn merge or switch)
    • Open any revision of any item.
    • View changes to any text files via integrated support for nine diff applications.
    • Disk cache for impressive speed.
    • Create printable `svn log` reports for any repository item or from the current log list.
  • Working Copy Inspector:
    • Smart, Flat & Tree views.
    • Item sorting/filtering/searching.
    • Operate svn actions on selected items.
    • Svn move/copy via drag & drop in Tree view.
    • Cleanup, ignore, rename, blame etc. commands via a contextual menu.
    • Svn update - to HEAD or any items to any revision.
    • Svn diff - any items against their base or previous revisions.
    • Svn log - view an item’s log and diff any 2 revisions.
    • Svn info, add, remove, revert, resolved, lock & unlock.
    • Interactive conflict resolving via your favourite diff application.
    • View, add, edit & delete Subversion properties for multiple items, simultaneously.
    • Automatic refresh preference.
    • Preference to call Subversion libraries directly - boosting refresh speed by up to 10 × as fast.
  • Save your working copies and repositories in a panel.
  • Activity window.
  • Review and Commit window:
    • Review highlighted changes while editing your commit message.
    • Mark which files you want to commit.
    • View & reuse recent messages.
    • Create & use powerful custom template messages & scripts.
