4. Quick Reference

Working copy window
return or enter If the file list is focused then opens selected files/folders.
any alpha-numeric key If the file list is focused then selects the next name beginning with that character.
⇧⌘F Reveal the selected file in the Finder.
⌘R Refresh display.
⌥⌘R Refresh display showing any repository updates.
⌘U Update the working copy to HEAD.
⌥⌘U Open the custom update sheet.
⌘D Diff the selected files against their pristine (BASE) copies.
⇧⌘D Diff the selected files against their previous (PREV) revision.
⌘L Open a sheet to browse the selected item’s log or diff it against any previous revision.
⌘S Open a repository window for this working copy.
⌃⌘T Switch to tree view.
⌃⌘F Switch to flat view.
⌃⌘S Switch to smart view.
double-click Item in file list to open it.
⌘P Open/Close the Subversion properties palette.
⌥⌘O Open/Close the Output drawer window.
drag Item from file list to an application to open it or to a Finder window to move it.
drop Drop a repository item, a log entry or a log entry path to a Working Copy window to merge.
Drop a repository directory while pressing ⌘ to switch.
right-click or ⌃-click list Opens contextual menu whose commands act upon the selected items.
Press ⌥ to reveal some alternate commands.
Repository window
double-click repository item Browse that item “as a sub-repository”.
⌘D (or ⇧⌘D) Diff the selected repository items log entry or log entry path against its previous (PREV) revision.
⌘B Open the ‘svn blame’ output for the selected files.
⌥⌘B Open the ‘svn blame --verbose’ output for the selected files.
⌘P Open log report settings sheet for selected item or current log items.
⌘K Checkout the selected directory.
⌘S Export the selected items.
⌘O or enter Open the selected items in appropriate applications.
⌥⌘O Open/Close the Output drawer window.
⌘R Refresh/update the log list.
⌥⌘R Refresh the browser view.
⌘L Open a diff/log sheet for the selected repository file/directory or log entry path.
⌘T Toggle between Advanced & Simple log displays.
space Browse the repository at selected log item’s revision.
drag Drag repository items to an application to open them, or to a Finder window to export or checkout.
Drag a repository item, a log entry or a log entry path to a working copy window to merge, or a repository directory while pressing ⌘ to switch.
drop Drop a file or folder (from the Finder) on to a directory in the repository browser to import it.
Review and Commit window
⇧⌘A Check all items in the available items list.
⇧⌘N Uncheck all items in the available items list.
⌘R Refresh the working copy & available items list.
⌘D Diff the selected file against its pristine copy.
⌘O or double-click Open the selected file.
⌘T Toggle commit status of the selected file.
⇧⌘F Reveal the selected file in the Finder.
⌘I Insert the currently selected recent message or template into message text.
drag File from list to an application to open it or to a Finder window to copy it.
Template keywords <MACHINE>, <USER>, <DATE>, <FILES>, <FILES>…</FILES> & <SCRIPT>…</SCRIPT>
Working Copies & Repositories windows
any alpha-numeric key Selects the next name beginning with that character.
⌘E Hide/show edit fields.
double-click or return Open or activate window for selected Working Copy or Repository.
⌥-double-click or ⌥-return Open new window for selected Working Copy or Repository.
drag Drag items up & down lists to move them. Press ⌥ to make a copy.
drag Drag items between windows to copy name, user & password.
drop Drop a folder in either window to create a new item.
Repositories window also accepts URLs & weblocs.
